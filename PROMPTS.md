# AI Prompt Used

Create a modern "Get in Touch" section for a technology conference website called DevConf 2026 using only HTML and CSS.

Requirements:
- Two-column layout
- Left side should contain a heading and a short description encouraging users to contact the event team
- Right side should contain a contact form with Name, Email, Message fields and a Submit button
- Use clean and semantic HTML
- Use modern CSS with Flexbox
- Add rounded corners, spacing, hover effects and professional colors
- The section should match the overall design of a modern tech conference website.
